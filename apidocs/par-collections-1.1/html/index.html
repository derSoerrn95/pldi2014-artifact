par-collections-1.1: Generic parallel combinators for data and iteration spaces.

These modules provide means to traverse and consume common data structures, in parallel, using any Par monad.

Additionally, a variety of parallel looping constructs are provided by this package (e.g. parallel traversals where the data structure is merely a range / iteration space.)
